## About me

I am a developer and data professional focused on building practical solutions with **Data Engineering, Artificial Intelligence and Full Stack Development**.

My work connects software development, data pipelines, applied AI and infrastructure. I have experience with **RAG systems, vector databases, data lakehouse architecture, backend/frontend development, computer vision and application deployment**.

Currently, I am deepening my research and practical work in **Information Retrieval, RAG evaluation, intelligent systems, data platforms and applied machine learning**.

---

## Main focus

- Data Engineering, ETL/ELT and data pipelines
- Data Lakehouse, Parquet, Raw/Processed/Curated layers
- Retrieval-Augmented Generation, Information Retrieval and vector search
- AI applied to real-world systems
- Computer Vision and image processing
- Full Stack applications with Angular, NestJS and FastAPI
- Cloud, containers, VPS deployment and application infrastructure

---

## Featured areas

### AI, RAG and Research

Development of pipelines for document ingestion, chunking, embeddings, vector storage and RAG-based search.  
Research involving Information Retrieval metrics, hybrid search, intelligent systems and RAG evaluation.

### Data Platform and Lakehouse

Construction of data workflows using APIs, relational databases, Parquet, analytical modeling and business KPIs.  
Focus on transforming raw data into structured insights.

### Full Stack Development

Development of complete systems using frontend, backend, APIs, authentication, dashboards and database integration.

### Deploy and Infrastructure

Experience with Docker, Portainer, Traefik, Linux VPS, Oracle Cloud and AWS for deploying applications and data services.
